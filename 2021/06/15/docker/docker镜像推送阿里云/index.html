docker镜像部署到阿里云

2021-06-15
2021-06-12
本文字数: 125 阅读时长: 1 分钟
阅读次数次

生成镜像

- docker build -f filename -t name:version .
- docker commit -a=’xxx’（作者） -m=’xxxx’（命名空间） 容器id 名字:版本号

docker push步骤（可参考阿里云文档）

- 1.创建镜像仓库
- 2.docker login --username= registry.cn-hangzhou.aliyuns.com
- 3.docker tag imageid registry.cn-hangzhou.aliyuns.com/仓库名/name:version
- 4.docker push registry.cn-hangzhou.aliyuns.com/仓库名/name:version
- 5.在阿里云镜像仓库查询是否有镜像（仓库名/name:version)

docker pull 拉取镜像

- docker pull registry.cn-hangzhou.aliyuns.com/仓库名/name:version
